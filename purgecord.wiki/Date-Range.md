# Date Range

Delete only messages posted between two points in time.

Set an **after** date, a **before** date, or both. Only messages inside that window get deleted.

One catch: date range and [Message Interval](Message-Interval) don't work together. If you set a message interval, the date filter gets ignored. Pick one or the other.

Useful when you want to wipe, say, everything from last year but keep the recent stuff.
