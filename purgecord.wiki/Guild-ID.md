# Server / Guild ID

The ID of the server the messages live in. "Guild" is just Discord's internal word for a server, same thing.

Click **current** next to the field while you're in the server and it fills itself in.

For DMs this gets set to `@me` automatically, that's Discord's way of saying "not a server, this is a direct message". You usually don't have to touch this at all, the buttons handle it.
