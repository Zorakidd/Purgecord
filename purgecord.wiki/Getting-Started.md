# Getting Started

The fastest path from "installed" to "channel is empty".

1. Go to the DM or channel in Discord you want to clear.
2. Click the 🗑️ in the top right. The panel opens.
3. Click the small buttons next to the fields one by one:
   * **me** next to Author ID (sets your own ID so only your messages get deleted)
   * **current** next to Server ID
   * **current** next to Channel ID
4. Optional: set filters if you don't want everything. See [Filters](Filters).
5. Hit **Delete**.
6. A confirmation with a preview shows up. If it looks right, confirm.

From there it runs through. You can hit **Stop** at any time.

Tip for your first run: try it on an old, unimportant DM, not your biggest channel right away. That way you get a feel for it without any risk.
