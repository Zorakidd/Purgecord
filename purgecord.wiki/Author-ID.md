# Author ID

This is the user ID of the person whose messages should be deleted. In almost all cases: your own.

Just click the **me** button next to the field and your ID gets filled in automatically.

Why this matters: in a server you normally only want to delete your own messages, not other people's (which you couldn't do anyway unless you have the permissions). Your Author ID makes sure only your stuff gets touched.

If you enter someone else's ID and you have the permissions to delete (mod in a server), you can clear their messages too. Without permissions, nothing happens.
