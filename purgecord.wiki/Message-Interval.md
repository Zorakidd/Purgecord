# Message Interval

Delete only messages between two specific messages, by their ID.

* **After a message:** click Pick, then click a message in the chat. Everything below it gets deleted.
* **Before a message:** click Pick, then click a message. Everything above it gets deleted.

Set both to delete only the range between two messages.

Note: if you use a message interval, the [Date Range](Date-Range) filter gets ignored. They don't combine.
