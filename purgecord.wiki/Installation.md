# Installation

## 1. Userscript extension

Nothing works without it. It manages the script in your browser.

* **Chrome / Brave / Edge / Opera:** [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/)
* **Firefox:** Tampermonkey, Violentmonkey or Greasemonkey

Small note for Chrome: since Manifest V3, Chrome sometimes gets fussy with userscripts. Brave usually runs smoother if you have the choice.

## 2. Install Purgecord

Open `purgecord.user.js` in the repo, click "Raw". Your extension catches it and asks to install. Confirm.

## 3. Check that it's there

Open Discord in your browser, go into a channel. There should be a 🗑️ icon in the top right. If it's there, you're good.

If not, check [Troubleshooting](Troubleshooting).
